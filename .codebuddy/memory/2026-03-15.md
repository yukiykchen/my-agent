# 2026-03-15 工作记录

## 新增证据收集 MCP 工具和前端展示增强

### 新增文件
- `server/mcp-servers/screenshot/main.go` + `go.mod` — 截图 MCP 服务器，提供 `take_screenshot`（网页截图）和 `fetch_page`（增强版网页内容抓取）两个工具，使用 Rod 无头浏览器
- `server/mcp-servers/textcompare/main.go` + `go.mod` — 文本比对 MCP 服务器，提供 `compare_texts` 工具，使用 gojieba 分词 + 余弦相似度/Jaccard/LCS 三维度比对

### 修改文件
- `server/.mcp.json` — 注册 screenshot 和 textcompare 两个新 MCP 服务器
- `server/cmd/server/main.go` — 新增 `/api/screenshots/:filename` 静态文件路由和截图目录初始化
- `client/src/types/index.ts` — 新增 ScreenshotResult、TextCompareResult、FetchPageResult 类型
- `client/src/components/MessageBubble.tsx` — 支持 Markdown 图片语法渲染和图片点击放大灯箱
- `client/src/components/ToolCallBlock.tsx` — 截图工具显示图片预览，文本比对显示相似度进度条可视化，页面抓取显示结构化内容
- `client/src/App.css` — 新增截图预览、图片灯箱、相似度可视化等样式
- `.gitignore` — 添加 Go 二进制文件和 data 目录忽略规则

### 架构决策
- 截图通过文件 URL 传递（保存到 server/data/screenshots/），避免 base64 内联导致 token 爆炸
- 截图和增强抓取合并到同一个 MCP 服务器（共用 Rod 浏览器实例）
- 文本相似度使用纯 Go 实现，无外部服务依赖
